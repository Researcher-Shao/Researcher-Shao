# Hi there, I'm Sheng Yi Shao | AI Health Researcher 👋

### 🔬 研究使命 (Research Mission)
作为母婴健康领域的 **Chief Researcher**，我坚持从**第一性原理 (First Principles)** 出发，构建具备临床严谨性、医疗伦理性与人文关怀的 AI 体系。我致力于通过技术研究定义行业标准，确保 AI 在母婴健康这一高风险、高敏感领域的安全应用。

### 核心研究版图 (Core Research Roadmap)

- **AI 问诊端到端体验与 GSB 测评体系:**
  - **研究重点：** 建立 AI 健康回答的端到端（End-to-End）全链路测评标准，涵盖从用户提问意图识别到最后医学建议输出的闭环。
  - **方法论：** 引入并深度定制 **GSB (Good/Satisfactory/Bad) 测评体系**，针对母婴咨询场景制定精细化的质量评估分级。

- **AI 医疗伦理、安全与安全 Benchmark:**
  - **伦理边界：** 深入研究 AI 问诊中的医疗伦理、数据隐私及算法偏见，确保模型决策符合医学伦理准则。
  - **安全基准：** 正在构建针对母婴健康的 **Medical Safety Benchmark**，专注于产科、儿科的用药安全、禁忌症提示及高危信号拦截的压力测试。

- **有图谱结构的 RAG 与知识驱动 (Graph-RAG):**
  - **知识底座：** 统筹 50 万级权威医疗语料（指南、期刊、手册）及药品库，产出包含 **258 个产科疾病** 的深度知识图谱。
  - **架构创新：** 通过 **Graph-RAG** 架构，将权威医学逻辑深度嵌入 LLM 决策过程，显著消除医学幻觉。

- **母婴 Agent 前沿研究 (Frontier MCH Agents):**
  - **前沿探索：** 研究具备长期记忆与主动关怀能力的 **MCH Agent**，探索多 Agent 协作在复杂育儿场景、婴儿粪便分析及个性化健康管理中的应用。
  - **仿真模拟：** 利用 **CERS (临床-情感双维度评分系统)** 与“真实患者模拟系统”驱动 Agent 进化，实现从“技术可用”到“医学可靠”的跃迁。
